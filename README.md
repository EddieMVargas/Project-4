# Project-4
This project analyzes natural disaster risk across two U.S. states using data from the Federal Emergency Management Agency National Risk Index (NRI). The goal is to compare FEMA’s standard risk calculation with a newly defined, independent risk model to identify potential bias in how risk is measured.
Method
Data: NRI + SVI
Tools: Python (Pandas, GeoPandas)
Custom Risk 
Results
NRI emphasizes economic loss
Custom model increases risk in vulnerable areas
Risk classification changes with weighting
Conclusion
Risk depends on how it is defined. Including social vulnerability provides a more balanced view.
